# Coach Notion Command Center - Template Blueprint

Use this blueprint to build the Notion template once, then sell repeatedly.

## Top-Level Pages
1. Home Dashboard
2. Leads & Clients (CRM)
3. Calls & Sessions
4. Programs & Offers
5. Content Studio
6. Revenue & KPIs
7. Weekly CEO Review
8. SOP Vault

## Database 1: CRM (Leads + Clients)
Properties:
- Name (title)
- Type (Lead/Client)
- Source (IG, referral, email, etc.)
- Stage (New, Contacted, Call Booked, Proposal, Won, Lost)
- Offer Interest
- Discovery Call Date
- Next Follow-Up
- Monthly Value
- Notes

Views:
- Pipeline (board by Stage)
- This Week Follow-Ups (calendar/list)
- Active Clients (filtered)

## Database 2: Calls & Sessions
Properties:
- Session Name (title)
- Client (relation to CRM)
- Session Date
- Session Type (Discovery, Coaching, Check-in)
- Agenda
- Notes
- Action Items
- Next Session Date

Views:
- Upcoming Sessions (calendar)
- Action Items Pending (table filter)

## Database 3: Programs & Offers
Properties:
- Offer Name
- Type (1:1, Group, Course)
- Price
- Delivery Length
- Status (Draft, Active, Paused)
- Capacity
- Enrolled Count
- Launch Date

Views:
- Active Offers
- Launch Timeline

## Database 4: Content Studio
Properties:
- Content Title
- Platform (IG, YouTube, TikTok, Email)
- Pillar (Authority, Story, CTA, Education)
- Status (Idea, Draft, Scheduled, Published)
- Publish Date
- CTA
- Linked Offer

Views:
- Content Calendar
- By Pillar
- Publish This Week

## Database 5: Revenue & KPIs
Properties:
- Month
- Revenue
- New Clients
- Discovery Calls
- Conversion Rate
- Avg Client Value
- Churn Count
- Notes

Views:
- Monthly KPI table
- Trend view (line chart if available)

## Weekly CEO Review Page
Sections:
- Wins this week
- Bottlenecks
- Top 3 priorities
- Metrics check
- Revenue goal vs actual
- Next week commitments

## SOP Vault
Suggested SOP pages:
- New lead response
- Discovery call prep
- Onboarding flow
- Session prep checklist
- Weekly content workflow
- Monthly review ritual

## Build Notes
- Keep visuals minimal and clean (neutral theme)
- Add instructional callouts for each section
- Include sample data so buyers understand setup quickly
