## 8) Scope Change Request SOP
1. Receive revision/change request in writing
2. Compare against agreed deliverables
3. Mark as in-scope or out-of-scope
4. If out-of-scope, send add-on quote + timeline
5. Wait for written approval before proceeding
6. Update task board and invoice if approved

## 9) Boundary Script (Out-of-Scope)
Hi [Client Name],

Great request—happy to help. This item falls outside our current scope. I can complete it as an add-on for [$X] with delivery by [Date]. If approved, I’ll add it to this week’s plan.
