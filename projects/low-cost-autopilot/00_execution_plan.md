# Low-Cost Autopilot Execution Plan

## Objective
Build revenue with **minimal upfront cash** and mostly autonomous execution.

## Budget Framework (starting)
- Phase 1 (now): **$0 required**
- Phase 2 (after traction): **$20-$30/mo** hosting/tools
- Phase 3 (scale tests): **$100-$300** optional ads (only if conversion proof exists)

## Chosen Lanes (low-cost first)

### Lane A — SellerPulse Lite (software lead magnet)
- Status: In progress (`projects/sellerpulse-lite`)
- Monetization path:
  1. Free analyzer tool
  2. Paid Pro pack ($19-$39 one-time)
  3. Optional optimization subscription ($49-$149/mo)
- Why now: fastest software path with near-zero infra cost and direct tie to existing Etsy audience.

### Lane B — Etsy B2B template packs (cashflow engine)
- Status: Day 1 assets created, Day 2 optimization running
- Monetization path:
  1. Starter templates ($7-$12)
  2. Pro bundles ($19-$29)
  3. License packs ($39-$59)
- Why now: proven channel + low production cost.

### Lane C — Crypto strategy (experimental only)
- Status: research only
- Rule: no live deployment until risk policy + budget approved.

## 14-Day Autopilot Sprint

### Days 1-3
- [x] Build SellerPulse Lite MVP
- [x] Add CSV import/export + downloadable plan
- [ ] Create simple landing page with email capture CTA
- [ ] Link from Etsy thank-you files to free tool

### Days 4-7
- [ ] Publish 3 starter Etsy listings from selected niches
- [ ] Publish 3 pro listings + 3 license listings
- [ ] Apply cross-sell matrix + coupon flow
- [ ] Launch weekly KPI tracker and baseline

### Days 8-10
- [ ] Release SellerPulse Pro paid download
- [ ] Add checkout + fulfillment flow
- [ ] Create 2 outreach posts/day for traffic (organic)

### Days 11-14
- [ ] Review KPI deltas
- [ ] Double down on top converting listing/product
- [ ] Prepare tiny paid test plan (only if metrics justify)

## Success Criteria (first 30 days)
- 20+ paid product sales (Etsy + Pro pack)
- 100+ free tool users
- 10+ qualified leads for recurring optimization offer

## Decision Gates
- If Etsy CVR <2% after optimization: rework offer positioning before ads.
- If SellerPulse landing conversion <15%: improve value prop + CTA before scaling traffic.
- If no clear winner by day 14: kill weakest lane and focus on top performer.
