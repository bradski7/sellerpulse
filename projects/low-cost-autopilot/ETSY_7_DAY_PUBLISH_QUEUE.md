## 7-Day Checklist

### **Day 1 (Pet Sitter - publish 3 listings)**
- [ ] Publish: Pet Sitter Starter ($7), Toolkit ($19), Ops Pack ($39)

- [ ] Update titles/tags with local-intent keywords (“pet sitter forms”, “dog walking business templates”)

### **Day 2 (Airbnb Cleaner - publish 3 listings)**
- [ ] Publish: Airbnb Cleaner Starter ($7), Toolkit ($19), Ops Pack ($39)
- [ ] Add bundle savings note in description (“buy Starter + Toolkit, save time on onboarding”)
- [ ] Coupon: **CLEAN10** (10% off, 5 days)
- **KPI checkpoint:** 40+ total shop visits, 3+ favorites, at least 1 add-to-cart across shop

### **Day 3 (Mobile Detailer - publish 3 listings)**
- [ ] Publish: Mobile Detailer Starter ($7), Toolkit ($19), Ops Pack ($39)
- [ ] Use consistent thumbnails (same brand style across all 9)
- [ ] Coupon: **DETAIL10** (10% off, 5 days)
- **KPI checkpoint:** 60+ total visits, 5+ favorites, conversion ≥ 1%

### **Day 4 (SEO + image optimization pass)**
- [ ] Update top image + first 160 chars for all 9 listings
- [ ] Add 2 long-tail tags per listing based on search terms report
- [ ] Keep one shop-wide coupon active: **LAUNCH15** (15% off orders $25+)
- **KPI checkpoint:** CTR on edited listings up by +20% vs Day 1–3 baseline

### **Day 5 (Pricing & offer test)**
- [ ] A/B test price on low traffic listings:
  - Starter: $7 → **$6.50** (3 listings max)
  - Toolkit stays $19
  - Ops Pack stays $39
- [ ] Add “compare at” language in description (time saved / client-ready docs)
- **KPI checkpoint:** 1+ sale by end of day OR add-to-cart rate ≥ 3%

### **Day 6 (Conversion boost)**
- [ ] Create targeted thank-you coupon auto-send to favoriters: **FAV20** (20% off, 48h)
- [ ] Add FAQ section to each listing (“editable?”, “instant download?”, “commercial use?”)
- [ ] Cross-link ladder in each description (“Start with $7, upgrade to bundle”)
- **KPI checkpoint:** favorites-to-orders conversion ≥ 5%, 2+ orders cumulative target

### **Day 7 (Scale winners + cleanup)**
- [ ] Identify top 3 listings by visits/favorites
- [ ] Duplicate winners into niche variants (e.g., cat sitter, turnover cleaning, ceramic coating)
- [ ] Pause underperformers (<10 visits + 0 favorites after 7 days) for rework
- [ ] Set ongoing coupon: **BUNDLE15** (15% off $25+)
- **KPI checkpoint:** Week target = 100+ visits, 10+ favorites, 2–5 sales, conversion 1.5–3%
