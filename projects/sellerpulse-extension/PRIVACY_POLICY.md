# SellerPulse for Etsy — Privacy Policy

Effective date: 2026-03-01

SellerPulse for Etsy is a Chrome extension that helps users open SellerPulse Analyzer from Etsy pages.

## Data we collect

The extension itself does **not** collect or sell personal data.

The extension stores only local settings in Chrome sync storage:
- connector URL
- default shop ID
- default sync days

## Data processed

When you choose to connect Etsy, the extension opens your configured SellerPulse Connector endpoint.
The connector (running locally or on your server) handles Etsy OAuth and API requests.

Any Etsy account data processing happens through:
- Etsy OAuth permissions you approve
- your own connector instance

## Data sharing

The extension does not send data to third-party analytics, ads, or tracking platforms.

## Security

Keep your connector credentials and token storage secure.
Do not publish secrets in extension files.
