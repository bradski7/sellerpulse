# Decision Framework (Low-Cost First)

## Rank opportunities with this priority

1. **Speed to first revenue** (days/weeks, not months)
2. **Startup cost** (prefer $0-$30/month)
3. **Automation potential** (can AI run 70%+ of workflow?)
4. **Distribution leverage** (existing channel/audience > new channel)
5. **Operational complexity** (support burden, compliance risk)

## Suggested lane order for Brad-style goals

1. Etsy expansion + product ladder
2. Software adjunct tied to Etsy pain (analytics, listing optimizer, template ops)
3. Service wrappers with productized delivery
4. Trading/speculative only as capped experiment

## Kill criteria (early)

Kill or pause if after one focused sprint:
- traffic exists but conversion remains weak after one major rewrite,
- support load > profit potential,
- legal/compliance uncertainty blocks fast execution,
- requires sustained paid traffic before proving conversion.
